API Manual

  • Select an HTTP method from the dropdown menu.
  • Enter the resource in the text input field.
  • Enter the request body in JSON format in the text area (optional).
  • Click the "Send Request" button to submit the request.
  • View the HTTP method, resource, request body, and HTTP response in the designated div elements.
  • Click the "Field Clearance" button to clear the fields.
  • The API manual is displayed in the "resourceList" div, providing instructions on how to retrieve all players, retrieve all teams, retrieve a player by team ID, create a new player, and delete a player.
  • GET all players:
    "https://student.csc.liv.ac.uk/~sgpgezah/v1/Rest.php?resource=players"

  • GET all teams:
    "https://student.csc.liv.ac.uk/~sgpgezah/v1/Rest.php?resource=players"

  • GET a player by team_id:
    "https://student.csc.liv.ac.uk/~sgpgezah/v1/Rest.php?resource=players&team_id=&player_id="

  • POST a new player:
    "https://student.csc.liv.ac.uk/~sgpgezah/v1/Rest.php?resource=add_player"
    • { "surname": "James", "given_names": "LeBron", "nationality": "American", "date_of_birth": "1984-12-30", "team_id": 1 }

  • DELETE player :
    "https://student.csc.liv.ac.uk/~sgpgezah/v1/Rest.php?resource=delete_player"
    • { "team_id": "", "id": "" }
  • UPDATE a player by team_id:
    "https://student.csc.liv.ac.uk/~sgpgezah/v1/Rest.php?resource=update-player&team_id=&id="
    • { "id": 1, "surname": "James", "given_names": "LeBron", "nationality": "American", "date_of_birth": "1984-12-30", "team_id": 1 }
